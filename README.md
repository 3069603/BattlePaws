## 📦 Deployment

To deploy via GitHub Pages:

1. Push this repository to GitHub.
2. Go to **Settings** → **Pages**.
3. Choose the main branch and `/root` as source.
4. Your site will be live at: `https://yourusername.github.io/repository-name/`
